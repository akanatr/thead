## Usage
```
thead [OPTIONS...] [FILEs...|DIRs...]  
　OPTIONS:  
    -c, --bytes <BYTES>　を出力する．
    -n, --lines <LINES>　
    -q, --quiet
    -s, --squeeze-blank連続した空行を1行にする．  
    -h, --helpこのメッセージを出力する.  
    
　ARGUMENTS   
　　FILEs... 
　　DIRs... 
```
